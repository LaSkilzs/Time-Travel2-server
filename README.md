## TimeTravel App Api

This is the api for the timetime travel app. The api features

1. Authorization/Authentication
2. Serialization
3. Pagination
4. Relational Database Design(normalization, associations)
5. Testing
6. API Versioning

# Technologies

1. Ruby
2. Rails
3. Rspec
